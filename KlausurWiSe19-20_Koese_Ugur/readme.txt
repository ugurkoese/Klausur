Allgemein:
- 2 Views -> loginView & supersecretView
- loginView: 2 TextFields -> username & password
			 1 Button -> signin / login
-supersecretView: orange
				  1 Label -> "SUPERSECRET"

Aufgaben: 
- username + password persistent speichern und supersecretView modal öffnen
- 3 login Versuche danach Button + TextFields für 10s disablen
- Label in supersecretView wird max. 20s angezeigt
- Button text soll sich von signin zu login falls man schon eingeloggt ist
- man kann mit cmd + shift + h die app verlassen (app nun im background), beim wiederöffnen der app soll die loginView angezeigt werden, sodass man sich nochmal anmelden soll
- wenn man schon angemeldet ist und die app wieder im foreground ist, soll ein Label "Welcome Back" angezeigt werden
